Installazione modulo Python per Chirp V2.9R3.10b
---------------------------------------------------------------------

- Aprire CHIRP

- Assicuratevi di essere in "Modalità sviluppatore",
	quindi -> menu Aiuto -> spuntare "Modalità sviluppatore"
	Al messaggio di allerta rispondete Sì e riavviate CHIRP come richiesto.

- Cliccare FILE in menu.

- Selezionare la voce "Carica Modulo".

- Al messaggio di allerta, rispondete Sì.

- Caricare il modulo allegato "uvk5_IJV_vx.xxxx.py".

- Leggere la radio con la normale procedura, selezionate nella lista delle radio il modello: "UV-K5-IJV vx.xxxx".

- N.B.: Il plugin "uvk5_IJV_vx.xxxxx.py" va caricato ogni volta che si vuole modificare la radio o il file .img.




Python module installation for Chirp
---------------------------------------------------------------------

- Open CHIRP

- Make sure you are in "Developer mode",
then -> Help menu -> check "Developer mode"
At the warning message answer YES and restart CHIRP as prompted.

- Click FILE in the menu.

- Select "Load Module".

- At the alert message, answer Yes.

- Load the module 'uvk5_IJV_vx.xxxxx.py'.

- Read the radio using the normal procedure, select the model: 'UV-K5-IJV vx.xxxx' in the radio list.

- N.B.: The plugin 'uvk5_IJV_vx.xxxxx.py' must be loaded every time you want to edit the radio or the .img file.
